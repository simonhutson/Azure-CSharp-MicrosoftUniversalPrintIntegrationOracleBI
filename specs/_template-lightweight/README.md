# Lightweight spec template

Copy this folder to a new numbered increment (for example `003-my-feature`) and replace all
placeholders.

Suggested flow:

1. Copy folder to `specs/NNN-<slug>/`.
2. Fill out `spec.md` with requirements and success criteria.
3. Fill out `plan.md` with implementation approach.
4. Break work into dependency-ordered items in `tasks.md`.

This template is intentionally minimal. Add a `contracts/` folder only when the feature needs
code-level implementation contracts.
